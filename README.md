# CodeLeap Network

React | TypeScript | Vite

A simple social feed application where users can create, edit and delete posts.

The application was built as a front-end challenge using React and TypeScript.

## Technologies

- React
- TypeScript
- CSS Modules
- Vite
- Context API
- React Router

## Getting Started

Clone the repository

git clone https://github.com/eummilena/codeleap-test.git

Install dependencies

npm install

Run the project

npm run dev
